# CleanMel+


**CleanMel+** is a PyTorch-based speech enhancement model that enhances logMel spectrograms from noisy speech using **Mamba** (a state-space model) for full-band sequence modeling. It improves upon the original [CleanMel](https://github.com/Audio-WestlakeU/CleanMel) (IEEE TASLP 2025) by replacing cross-band linear projection with full-band Mamba, enabling the model to capture temporal dynamics and long-range spectral coherence.
